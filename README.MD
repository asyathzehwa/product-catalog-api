# Product Catalog API

## Description
Simple Node.js & Express API that displays product information.

## Project Structure
product-api/
 ├── server.js
 ├── public/index.html
 ├── package.json
 ├── .gitignore

## API Endpoints
### GET /products
Returns all products.

### GET /products/categories
Returns unique categories.

### GET /products/instock
Returns only in-stock products.

## Installation
1. Clone repo
2. Run: npm install
3. Run: node server.js
4. Visit: http://localhost:3000
